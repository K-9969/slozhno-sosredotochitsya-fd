## Первая проверка проетной работы
### Проведеные работы:
1. Создан лейаут странцы, блоков и элементов, с использованием grid и flex контейнеров
2. Добавлены декоративные элементы.
3. Добавлены отступы между элементами и блоками.
4. Изменен шрифт на всей страницы, стиль начертания изменен в зависимости от необходимости непосредственно у элементов.
5. Заданы и переопределены переменые для разных тем.
